RetailDeploy Framework

The RetailDeploy Framework is contained in a .MSI package. The instructions to deploy it are as follows:

Source location

  • {{release-folder}}\Deploy

Files

  • RetailDeploy.msi

Destination location

  • D:\Data\INETPUB\Installs

UAT servers

  • SV232104 (UAT.STG WEB & FILE & Analytics)
  • SV232105 (UAT.WEB.LIVE)
  • SV232103 (UAT.FILE.LIVE)
  • SV206051 (UAT.PIS.LIVE )

PROD servers

  • SV204744 (PROD.STG WEB& FILE )
  • SV204745 (PROD Analytics)
  • RETAILWEB 1/2/8 (PROD.WEB.LIVE)
  • SV237970 (Additional PROD.WEB.LIVE WS12)
  • RETAILWEB 3/6 (PROD.FILE.LIVE )
  • RETAILWEB 4/5 (PROD.PIS.LIVE)
  • SV180611 (PROD DR Analytics)
Deployment steps
  1. Delete all content (files and directories) from D:\Data\INETPUB\RetailDeploy\
  2. Copy File from Source location to Destination location (this will overwrite the previous version)
  3. Open cmd as administrator and navigate to Destination location
  4. Execute RetailDeploy.msi
